# CS549Project3 Spark Project

Problems 2 and 3 Mario Arduz

## compile

sbt compile

## run

sbt "run [1-4]"

where [1-4] is the problem number

## Notes
Add to bashrc if needed
export SPARK_LOCAL_HOSTNAME=localhost


## Generating Data

Customer.txt and Transcations.txt are taken from project 1

Points.txt is already generated. If not then execute this command

sh Problem3/start.sh;cp points.txt ../data

### Matrix
matrix1.csv and matrix2.csv exist in data/ as 500x500 matricies.

matrix1.csv and matrix2.csv can generated by passing the arguement "True" when running problem2

sbt "run 2 True"
